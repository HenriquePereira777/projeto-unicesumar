Site responsivo de portfólio pessoal apresentando minha trajetória como Desenvolvedor Backend e Analista de Dados, com foco em Java, Spring Boot, APIs REST e Business Intelligence.

## 🗂️ Estrutura de Páginas

| Página | Descrição |
|---|---|
| `index.html` | Página inicial com hero animado e foto |
| `sobre.html` | Apresentação, habilidades técnicas por categoria |
| `projetos.html` | Cards dos projetos desenvolvidos |
| `contato.html` | Formulário de contato com validação JavaScript |

---

## ✨ Funcionalidades

- ✅ Layout responsivo — adapta para mobile, tablet e desktop
- ✅ Animação de digitação no cargo (efeito typewriter)
- ✅ Partículas animadas no fundo via Canvas API
- ✅ Dark/Light mode com persistência via localStorage
- ✅ Formulário de contato com validação JavaScript inline
- ✅ Efeito 3D no nome ao passar o mouse
- ✅ Foto flutuante com animação CSS
- ✅ Header com efeito blur e animação no scroll
- ✅ Animação de sublinhado nos links do menu
- ✅ Cards de projetos com hover interativo
- ✅ SEO básico com meta tags em todas as páginas
- ✅ Favicon personalizado com logo HP

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica das páginas
- **CSS3** — Estilização, animações, variáveis CSS e responsividade
- **JavaScript** — Interatividade, validação e efeitos dinâmicos
- **Canvas API** — Partículas animadas no fundo
- **localStorage** — Persistência do tema dark/light
- **GitHub Pages** — Hospedagem gratuita
